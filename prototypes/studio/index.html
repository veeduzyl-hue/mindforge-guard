Local-First Review Workspace

Studio Import / Export Prototype

This prototype shows the first bounded consumer surface for Evidence Packs, Governance Reports, Markdown / HTML renders, and Evidence Index exports.

It is downstream-only: Studio consumes Guard Core outputs and does not compute verdicts, risk, coverage, or reason codes on its own.

The interactive shell here is static by design. The actual orchestration lives in prototypes/studio/studioWorkflow.mjs.

Import Evidence Pack

Local Input Only

Select or paste local pack JSON. The browser prototype shows the import step conceptually only. A reviewer can choose a local file or paste pack JSON for the Node workflow helper. Studio does not send packs over a network, does not persist them outside the local workspace, and does not inspect artifact file contents.

Intended thin entrypoint: pass a JSON string or parsed pack object into runImportedPack(...).
