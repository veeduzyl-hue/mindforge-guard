Local-First Review Workspace

Studio Local Browser Wiring

This prototype wires local browser-side file selection, paste JSON input, and download controls into the existing Studio review surface without moving governance logic into the browser.

Guard Core remains the governance source of truth. The browser UI only captures local input, updates local status, and saves already-loaded output strings when they are present.

The browser prototype keeps all input in memory only. Actual report generation is delegated to runImportedPack(...) and Guard Core in the local Studio runtime.
