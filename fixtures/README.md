# Evidence Schema v1 Fixtures

These fixtures are synthetic local Evidence Pack inputs for MindForge Guard.

They are not governance reports.
They do not contain governance verdicts.
They do not grant approval authority.
They do not imply that Harness computes governance verdicts.

Guard Core will later compute coverage, risk, verdict, receipt, and report model.
Harness remains a producer only.

These fixtures are intended to support upcoming PR-05 parser and validator work and PR-08 report service development.

Included scenarios:

- `ai-pr-low-risk-complete`
- `ai-pr-missing-tests`
- `dependency-upgrade-breaking-change`
- `release-prep-missing-rollback`
- `cyber-remediation-authorized-patch`

All fixture data is synthetic, local, and non-sensitive.
