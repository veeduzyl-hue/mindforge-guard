intended task:
Summarize low-risk HR self-service requests and route them for human review.

in-scope behavior:
Read request metadata, summarize the request, and prepare a bounded response draft.

out-of-scope behavior:
Do not approve policy exceptions, modify payroll records, or execute external changes.

success criteria:
Produce a reviewable summary with traceable inputs and bounded action context.

known limitations:
The preview pack uses synthetic request data and does not cover production integrations.
