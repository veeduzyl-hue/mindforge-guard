# Governance Evidence Profile Boundary

`v5.0` phase 1 introduces the governance evidence layer as a consumer-facing,
recommendation-only profile above policy stabilization.

Frozen boundaries:
- recommendation-only
- additive-only
- non-executing
- default-off
- provenance preserving
- lineage preserving

Preserved semantics:
- policy stabilization semantics preserved
- enforcement semantics preserved
- approval semantics preserved
- judgment semantics preserved
- permit gate semantics preserved
- audit output preserved
- audit verdict preserved
- actual exit code preserved
- denied exit code preserved at `25`
- no governance object addition
- no authority scope expansion
