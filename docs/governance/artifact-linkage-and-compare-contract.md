# Artifact Linkage And Compare Contract

The phase-2 governance evidence contract freezes:

- artifact linkage contract
- replay readiness profile
- compare compatibility contract
- evidence receipt-readiness surface
- evidence consumer compatibility surface

The contract guarantees:
- recommendation-only posture
- additive-only posture
- non-executing posture
- default-off posture
- replay readiness without execution
- bounded artifact linkage
- compare compatibility without takeover
- audit output / verdict / exit preservation
- deny exit code preserved at `25`
- no governance object addition

The contract does not provide:
- execution authority
- automatic replay
- default-on enablement
- authority scope expansion
- governance object creation
