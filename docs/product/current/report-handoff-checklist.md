# Report Handoff Checklist

## Before You Share

Confirm the packet is ready for a human reviewer.

## Required Files

- Evidence Pack folder
- guard-pack-validate.json
- guard-single-agent-report.json
- optional Markdown summary
- known missing evidence
- security review notes when needed

## Reviewer Context

Include:

- workflow name
- workflow owner
- technical owner
- review owner
- intended reviewer
- review purpose

## Missing Evidence Notes

List:

- missing files
- missing run evidence
- redacted evidence
- stale evidence
- evidence intentionally omitted

## Security / Redaction Notes

Do not share secrets, credentials, tokens, or private keys.

Use synthetic, redacted, or summary evidence for sensitive material.

## Follow-up Questions

Include:

- what the reviewer should inspect
- what decisions remain outside Guard
- what evidence should be added before reuse

## Boundary

The human reviewer decision remains outside Guard.

MindForge Guard does not approve, block, deploy, certify, guarantee legal compliance, or control execution.
